# TypeScript Software Engineering Roadmap

A structured learning path from TypeScript basics to advanced software engineering concepts.

## Prerequisites

- [Node.js](https://nodejs.org/) (v18+)
- [TypeScript](https://www.typescriptlang.org/) (`npx tsc --version`)

## Getting Started

```bash
npm install
```

Run any lesson:

```bash
npm run basics:01   # Variables
npm run basics:02   # Data Types
# ... or run all basics at once:
npm run basics:all
```

Type-check the project:

```bash
npm run typecheck
```
